Author Preferences

Use the menu on the left to select the types of preferences you would like to amend.

Default Filter Settings

Your default preferences are applied where you have no specific distribution preferences, such as those distributions that you have recently uploaded, or if you wish to have some or all of your distributions use the same set of preferences.

Distribution Filter Settings

Upon selecting this option, a list of the distributions known to have been released by you will be listed. Note that this page can take several minutes to load, as it reads multiple indices to verify the definitive list of distributions. Select a distribution and apply the preferences you would like to have applied when a report for that distribution is received.
